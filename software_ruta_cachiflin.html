Políticas de Privacidad y Uso

1. Uso del Sistema

El Sistema de Control Logístico Ruta Cachiflin está destinado exclusivamente al uso interno del personal autorizado. El acceso a la plataforma implica la aceptación de las presentes políticas. Cualquier uso indebido, acceso no autorizado o divulgación de información a terceros está estrictamente prohibido y podrá resultar en acciones disciplinarias o legales.

2. Confidencialidad de la Información

Toda la información gestionada en este sistema (rutas, incidentes, reclamos, contratos, datos de conductores y clientes) es propiedad exclusiva de la empresa. El personal autorizado se compromete a mantener la confidencialidad de dicha información, no compartirla fuera del entorno laboral y utilizarla únicamente para fines operativos legítimos.

3. Tratamiento de Datos Personales

Los datos personales recopilados (correo electrónico, registros de actividad, historial de operaciones) son tratados con base en una relación laboral legítima. Dichos datos son utilizados exclusivamente para la gestión logística interna, auditoría de procesos y mejora continua del servicio. No se venderán ni compartirán con terceros sin consentimiento expreso.

4. Uso de Cookies

Este sistema utiliza los siguientes tipos de cookies:

  • Esenciales (Obligatorias): Necesarias para el inicio de sesión seguro mediante Firebase Authentication y el control de acceso por roles. No pueden desactivarse.
  • Rendimiento y SLA: Permiten el refresco automático de indicadores de tiempo real (barras de SLA, alertas de ruta). Mejoran la experiencia operativa sin recopilar datos de navegación externa.
  • Preferencias de Interfaz: Guardan el último panel visualizado para que cada sesión inicie donde el usuario lo dejó.

Al aceptar estas políticas, el usuario consiente el uso de las cookies descritas anteriormente.

5. Derechos ARCO y Derecho al Olvido

De acuerdo con la normativa de protección de datos aplicable, todo usuario tiene derecho a:

  • Acceso: Conocer qué datos personales están almacenados sobre su persona.
  • Rectificación: Solicitar la corrección de datos inexactos o incompletos.
  • Cancelación (Derecho al Olvido): Solicitar la eliminación permanente de sus datos personales del sistema.
  • Oposición: Oponerse al tratamiento de sus datos en circunstancias específicas.

La eliminación de datos y cuentas de usuario es gestionada exclusivamente por el Administrador del Sistema. Para ejercer cualquiera de estos derechos, el usuario debe comunicarse directamente con el administrador a través de los canales internos autorizados por la empresa.

6. Responsabilidades del Usuario

Cada usuario es responsable de mantener la confidencialidad de sus credenciales de acceso. Está prohibido compartir contraseñas, iniciar sesión desde dispositivos no autorizados o intentar acceder a secciones del sistema que no correspondan a su rol asignado. El sistema registra automáticamente cada acción mediante logs de auditoría inmutables.

7. Actualizaciones de estas Políticas

La empresa se reserva el derecho de actualizar estas políticas cuando lo considere necesario. Los usuarios serán notificados de cambios significativos al iniciar sesión. El uso continuado del sistema implica la aceptación de las políticas vigentes.
